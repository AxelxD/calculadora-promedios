# Proyecto: Promedio y Calificación de un Alumno (Java)

Este proyecto es un programa en **Java** que solicita por teclado el **nombre del estudiante** y **5 calificaciones**, calcula el **promedio** y asigna una **calificación final en letra** (A, B, C, D, E o F) según el rango establecido.  
Finalmente, imprime en pantalla los resultados con el formato solicitado.

## Requisitos
- Java JDK 8 o superior
- Consola / Terminal (Windows, macOS o Linux)

El programa te pedirá:

El nombre del estudiante

Las 5 calificaciones (una por una)

Al final mostrará:

Nombre

Calificación 1 a 5

Promedio

Calificación final (letra)

Notas

Las calificaciones se guardan en un arreglo de tamaño 5.

El promedio se calcula sumando las 5 calificaciones y dividiendo entre 5.

La letra final se asigna con base en el promedio:

<= 50: F

51–60: E

61–70: D

71–80: C

81–90: B

91–100: A
